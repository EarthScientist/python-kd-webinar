# python-kd-webinar
This repository contains the code associated with the "Explore algorithms
in Python with a GUI-based app" Webinar held on August 19 2015

The files are discussed in the following order:

  1. region
  2. kd0
  3. app0
  4. kd1
  5. kd2
  6. app2
  7. app3

Naturally you can remove the iteration number if you choose to keep these
implementations for your own use.

All code examples assumes a standard installation of Python 3.
